Eventcus

Eventcus is an event management platform for event organizers to create, advertise, release and allocate tickets and track payments. To be used by at least 2 Imperial societies in the coming year with a collective user base of 200+. Made using React JavaScript, Node.js, Google App Scripts, MongoDB.

On the Dashboard page, adminstrators can see all of their past and future events on the calendar and in detail below. By clickling the dropdown on each event adminstrators can quickly see how many tickets have been sold and reserved.

Adminstrators can quickly create events with lots of customization options. Once an event is created a Google Form and Sheet will be avaliable to the admin.

Clicking on an event on the dashboard will take you to it's event page, here you can see more details of the event and admins can allocate tickets, set payment statuses and email the ticket holders.

On the email page adminstrators can verify, edit and batch send emails to different types of ticket holders.

Virtual Imperial

Virtual Imperial is a virtual version of the Imperial College London campus in Minecraft. It is intended for Freshers in the coming year to socialise when studying remotely. Using Java, MongoDB, Angular I created mods allowing lecturers to upload lecture links and schedules such that users in the virtual campus can go to the lecture hall and access the lecture’s links. We combined 3D postional audio such that users can use voice chat in the world for a more immersive experience.
